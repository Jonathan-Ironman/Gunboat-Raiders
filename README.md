# Gunboat Raiders

Speedboat combat game built with Three.js. Armed boats, broadside cannons, arcade action on the open sea.

## Competition Entry

Built for the [Three.js Water Pro Giveaway](https://threejswaterpro.com/) challenge.

## Tech Stack

- Three.js (React Three Fiber)
- Rapier Physics (react-three-rapier)
- Vite
- TypeScript
